A point is a 0-[[dimension]]al mathematical object which can be specified in $n$-dimensional space using an [[n-tuple]] ($x_1, x_2, ..., x_n$) consisting of $n$ [[coordinate]]s. 

[[Wolfram MathWorld|Wolfram]] adds:

> Although the notion of a point is intuitively rather clear, the mathematical machinery used to deal with points and point-like objects can be surprisingly slippery. This difficulty was encountered by none other than [[Euclid]] himself who, in his _[[Elements]]_, gave the vague definition of a point as "that which has no part."
