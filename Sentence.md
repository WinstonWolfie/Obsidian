A sentence is a logic formula in which every variable is [[quantified]].

[[Wolfram MathWorld]] says:

> The formula $\exists (x, \forall (y,y \in x)),$ in which each variable is quantified, can be translated into English as the complete sentence "There exists a set which has every set as an element." However, the formula $\forall (y,(y \in x)),$ in which $x$ is not quantified, can only be translated as the sentence fragment "Every set is an element of \_\_\_," where "\_\_\_" is unspecified because $x$ is not quantified.
